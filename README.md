# CompTIA Security+ (SY0-701) Exam Simulator

A simple, fast, and interactive web-based exam simulator designed to help you prepare for the CompTIA Security+ (SY0-701) certification. This simulator provides a realistic testing experience by generating a randomized 90-question exam from a comprehensive bank of over 330 questions.

## 📝 Acknowledgements / Credits

The questions, concepts, and study materials in this simulator were generated and based entirely on the excellent study notes provided here:  
**[Wilsonvs CompTIA-Security-SY0-701 All in One.md](https://github.com/wilsonvs/CompTIA-Security-SY0-701/blob/main/All%20in%20One.md)**

## ✨ Features

* **Realistic Exam Simulation**: Each time you start the exam, exactly 90 questions are randomly selected and shuffled from the master question bank to simulate the real CompTIA exam length and variability.
* **Anti-Memorization (Shuffled Options)**: The order of the multiple-choice and multi-select options is randomly shuffled every single time a question is displayed, ensuring you actually learn the concepts rather than memorizing option positions (A, B, C, D).
* **Large Question Bank**: Includes 331 questions covering all 5 core domains of the SY0-701 exam:
  * Domain 1: General Security Concepts
  * Domain 2: Threats, Vulnerabilities & Mitigations
  * Domain 3: Security Architecture
  * Domain 4: Security Operations
  * Domain 5: Security Program Management & Oversight
* **Instant Feedback & Explanations**: Review your answers at the end of the exam to see your score, track your performance per domain, and read detailed explanations for the correct answers.

## 🚀 How to Run

Because this is built with standard web technologies, there are no dependencies to install!

1. Clone this repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Click **Start Exam** to begin your randomized 90-question test session.
